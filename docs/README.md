## Что вообще происходит, в двух абзацах

Госуслуги отдают свободные слоты записи в МВД по ручке
`POST /api/lk/v1/equeue/agg/slots`. Она пускает только по кукам активной сессии,
поэтому сторож держит открытое окно настоящего Chrome, в котором вы один раз
залогинились руками, и выпускает запросы **изнутри страницы** — так они несут
подлинные куки, заголовки и TLS-отпечаток браузера.

Раз в ~8 минут (с разбросом, чтобы интервалы не были машинно-ровными) сторож
обходит выбранные отделения. Как только в ответе появляются слоты — поднимает
шум: голосом и звуком на самом Маке, а также в тех каналах, что вы включили.
Пока слоты не пропадали, повтор подавляется, чтобы не звонить каждый цикл.
